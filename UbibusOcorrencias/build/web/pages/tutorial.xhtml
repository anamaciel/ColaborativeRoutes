Esta página exibe um rápido tutorial mostrando os passos que devem ser seguidos para registrar novas ocorrências na aplicação.


Para realizar o registro de novas ocorrências é necessário que o usuário tenha efetuado login na aplicação.

Para efetuar login o usuário deve possuir um cadastro que pode ser realizado acessando a opção criar uma conta no menu principal.


Após ter realizado o cadastro basta informar o login e a senha no formulario de login disponível na parte superior esquerda da página principal da aplicação.


Depois de ter efetuado o login na aplicação o usuário está habilitado à registrar novas ocorrências.

Para registrar ocorências é utilizado o mapa disponível na página principal, bastando clicar com o mouse sobre o ponto do mapa referente a locolização da ocorrência que se deseja cadastrar.

DICA: Para melhorar a precisão da localização deve-se aumentar o zoom do mapa. Quanto maior o zoom mais precisa é a localização.

Ao clicar no mapa uma caixa de diálogo será exibida. Nela estão presentes os campos que devem ser preenchidos com os dados da ocorrência.


Depois de preenchidos os dados basta clicar no botão

Feito isto a caixa de diálogo irá desaparecer e um marcador aprecerá no mapa no local exato onde foi adicionada a ocorrência.

Agora os dados desta ocorrência já podem ser visualizados por qualquer usuário, bastando clicar sbre o marcador, como mostra a figura abaixo.

Lembrando que apenas as ocorrências que ainda não foram resolvidas são exibidas no mapa.
